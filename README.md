## NotesApp

Sovelluksen avulla voit tallentaa muistiinpanoja eri opintojaksoille.

Päänäkymästä voit valita, haluatko:
- Lisätä uuden opintojakson
- Lisätä uuden muistiinpanon (toimii vain, jos opintojaksoja on olemassa)
- Katsoa ja poistaa olemassaolevia muistiinpanoja

## Kehitysversion käynnistäminen
1. Aja `npm install` asentaaksesi riippuvuudet
2. Aja `npm run dev`
